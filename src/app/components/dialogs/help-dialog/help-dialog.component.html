Basic instructions

Commands must be typed in the input bar. Commands within <> indicate an option (valid options are separated by the | character) to be entered after a space. Try some of the following commands:

/help - Shows this help message

/theme <green | cyan | amber> - Changes the application theme. If no theme is specified, it switches to the next one.

/clear - Clears the terminal startup output

/about - Shows information about this application

Press ESC to close any modal.
